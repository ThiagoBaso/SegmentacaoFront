# 🌱 Sistema de Segmentação de Talhões (Em Desenvolvimento)

Aplicação web para segmentação e análise de áreas agrícolas a partir de imagens, permitindo a identificação e manipulação de regiões (talhões) de forma interativa.

O sistema é dividido em dois repositórios independentes:

- **Front-end**: Interface interativa para visualização da imagem, seleção de pontos e exibição dos talhões segmentados  
- **Back-end**: API responsável pelo processamento da imagem, segmentação e retorno dos dados para o cliente  

---

## 🚀 Funcionalidades

- Upload de imagens para análise  
- Interação direta com a imagem (seleção de pontos)  
- Segmentação de áreas (talhões) com base nas entradas do usuário  
- Visualização dos resultados em mapa/imagem  
- Integração entre front-end e back-end via API  

---

## 🛠️ Tecnologias

### Front-end
- React  
- Leaflet (manipulação de mapas/imagens)  
- JavaScript   

### Back-end
- Python
- API REST  
- Processamento de imagem
